Odoo Mail Client - End User Quick Start

Use this guide to start reading, sending, and organizing email in Odoo within minutes.

1) Before You Start

  1. Log in with your Odoo user account.
  2. Make sure your administrator has configured at least one mail account for you.
  3. If your Inbox is empty, ask your admin to run a manual fetch.

2) Open the App

  1. Open the Odoo app launcher.
  2. Select Email Management.
  3. Use mailbox views: Inbox, Sent, Draft, Unread, Starred, Archived, All Emails.

3) Read and Process Messages

  • Open a message to read full details.
  • Use actions: Mark Read/Unread, Star/Unstar, Archive/Unarchive.
  • Use Reply and Forward directly from the message form.
  • Use search and filters to find emails quickly.

4) Send a New Email

  1. Create a draft message.
  2. Fill To (required), then Cc/Bcc if needed.
  3. Add subject, body, and attachments.
  4. Select From Account if multiple accounts exist.
  5. Click Send.
If account SMTP is configured, the message is sent through that account. Otherwise, Odoo may queue delivery based on configuration.

5) Organize Your Mailbox

  • Star important messages for follow-up.
  • Archive completed threads.
  • Apply tags to classify conversations.
  • Use priority to flag urgent items.

6) Troubleshooting Quick Checks

  • Cannot send: confirm at least one valid recipient in To.
  • SMTP error: re-check From Account and notify admin.
  • No incoming mail: refresh view and ask admin to fetch now.
  • Missing expected emails: verify correct user login; mailboxes are private by account/user rules.
Contact your administrator for account setup changes, permissions, or sync failures.
